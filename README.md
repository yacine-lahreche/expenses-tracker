## UI Principles
- **No-Line Rule**: Boundaries via color shifts and inner glows.
- **Glassmorphism**: High backdrop blur (24px+) and semi-transparent surfaces.
- **Deep Palette**: Atmospheric navies with neon primary/secondary accents.
- **Dual Typeface**: Manrope for headlines, Inter for data.
